bAbi

I thought it would be clever to solve bAbi, with the intention of creating my own dataset.

So with a deep interest in the subject in general and why my code was getting answers wrong in particular, I present a few minor nits to pick with the dataset (and a big one with task 2).

While most of these issues are rare, because this is synthetic data, errors are more serious, as they suggest problems with the underlying model.

Version of bAbi used: 1.2.

Links:
  • fb.ai/babi
  • Jason Weston, Antoine Bordes, Sumit Chopra, Tomas Mikolov, Alexander M. Rush, "Towards AI-Complete Question Answering: A Set of Prerequisite Toy Tasks", arXiv:1502.05698 [cs.AI].
  • Sainbayar Sukhbaatar, Arthur Szlam, Jason Weston, Rob Fergus, "End-To-End Memory Networks", arXiv:1503.08895 [cs.NE].
